# Vinay Kumar Rao Pikala — Terminal Portfolio

Interactive terminal-style portfolio built with React, Vite, and TypeScript.

## Getting started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```


## Commands

`help`, `whoami`, `skills`, `projects`, `experience`, `education`, `contact`, `resume`, `github`, `linkedin`, `clear`, `banner`

Toggle **Simple view** for a traditional layout.
